# Backlog do Programa

Backlog inicial do Programa de Eficiência Financeira e Governança Operacional do Setor Financeiro do Hospital.

## Como usar
- Cada item do backlog deve ter problema, objetivo, prioridade, responsável e próximos passos.
- A priorização deve considerar risco operacional, impacto no caixa, esforço e dependências.
- Revisão sugerida: semanal.
