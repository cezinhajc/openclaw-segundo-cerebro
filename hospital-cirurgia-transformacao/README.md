# Hospital Cirurgia — Programa de Transformação

Base de trabalho para estruturar o programa de transformação digital e eficiência operacional do Setor Financeiro do Hospital Cirurgia.

## Objetivo
Organizar contexto, diagnóstico, iniciativas, status reports, riscos, decisões e roadmap do programa.

## Estrutura
- `contexto/` — visão do hospital, do setor, pessoas, sistemas e desafios
- `programa/` — visão geral, objetivos, frentes, roadmap, riscos e decisões
- `operacao/` — status reports, backlog, iniciativas e rituais
- `dados/` — indicadores, BI e temas ligados a caixa e priorização
- `templates/` — modelos de input, status report e atas

## Observação
Esta base começa pelo Setor Financeiro, mas foi nomeada de forma ampla para poder crescer com o cliente sem reestruturar tudo depois.
