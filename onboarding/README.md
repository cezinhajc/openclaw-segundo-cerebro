# Onboarding — Configure o Cérebro da Empresa

Esta pasta contém o guia de setup para transformar esta base em um cérebro operacional de uma empresa real.

## Como começar

Clone ou faça fork do repositório e cole este comando no agente:

```text
Leia onboarding/SETUP.md e configure o cérebro da minha empresa
```

O agente conduz o processo inteiro. Tempo estimado: 60–90 minutos.

## O que o setup faz

1. limpa os dados de demonstração, mantendo a estrutura
2. preenche o contexto da empresa
3. configura a primeira área operacional
4. cria a primeira skill
5. agenda a primeira rotina
6. publica o cérebro no GitHub
